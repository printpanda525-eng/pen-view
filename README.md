# Pen View

Pen View is a lightweight Blender add-on that provides a floating popup with quick viewport and mode tools, designed for pen tablet users.

The add-on focuses on improving navigation and workflow in the 3D View when working without a full keyboard or numpad.

---

## Features

- Popup-based viewport view controls (Top, Front, Side, Bottom)
- Quick access to Object, Edit, and Sculpt modes
- Mesh selection mode buttons (Vertex, Edge, Face) in Edit Mode
- Overlay toggles for Wireframe, X-Ray, and Face Orientation
- Designed for pen tablets and laptops without numpad

---

## Usage

The add-on is accessed via a custom shortcut that opens a floating popup at the cursor position.
